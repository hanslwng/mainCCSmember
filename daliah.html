BloomCart In-Progress

BloomCart is a full-stack e-commerce web application designed for a flower shop, enabling users to browse, search, and order flowers online. The system features a dynamic product catalog, shopping cart functionality, and a secure checkout process. The front end is built using HTML, CSS, and JavaScript for a responsive user interface, while the back end is powered by Flask and MySQL to handle user authentication, order management, and real-time database interactions.
